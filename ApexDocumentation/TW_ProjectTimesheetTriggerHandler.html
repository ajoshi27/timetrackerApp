TW_ProjectTimesheetTriggerHandler

public class TW_ProjectTimesheetTriggerHandler implements TW_TriggerInterface
This class implements the TW_TriggerInterface. it contains the basic trigger event methods like beforeInsert, beforeUpdate beforeDelete etc. These methods will be fired on insert , update and delete events of TimeWatch__Project_Timesheet__c record.
