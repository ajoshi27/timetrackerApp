TW_ProjectMemberTriggerHelper

public class TW_ProjectMemberTriggerHelper
It contains the helper and utility methods for TimeWatch__Project_Member__c object trigger.All This method will be called from TW_ProjectMemberTriggerHandler.class. Every time new Project Member has been Added, methods of this class will fire. It has method to share project related time entries, timesheet, tasks with this new member. If new member is 'Manager' then update related project record with manager details.
